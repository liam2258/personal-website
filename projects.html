Offender Profile 2025

The Offender Profile is the premiere report for the Research & Records team at the Alaska DOC.

With the support of my supervisor I became the lead developer of this project and lead a modernization effort. The previous profiles were non-interactive PDFs that were manually assembled using image editing software. The 2025 Profile was completely redesigned to be an interactive web experience with numerous quality of life features like interactive graphs, downloadable aggregated data sets, better web accessibility for disabled folk and more!

The entire data collection process was pipelined using T-SQL and Python, while the report itself was developed using Quarto, a technical publishing system that combines Python, HTML/CSS, and Markdown to creative replicatable scientific publications!
